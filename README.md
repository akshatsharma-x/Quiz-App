# Quiz App

A full-stack **Quiz Application** that allows users to test their knowledge through interactive multiple-choice questions. The application provides a simple interface where users can answer questions and view their final score at the end of the quiz.

This project demonstrates basic **frontend and backend web development concepts**, including UI design, client-server interaction, and dynamic content rendering.

---

# Features

* Interactive **multiple-choice quiz**
* **Dynamic question loading**
* **Real-time score calculation**
* Clean and simple **user interface**
* Separation of **frontend and backend**
* Beginner-friendly **full stack project**

# How It Works

1. The **frontend displays quiz questions** to the user.
2. Users select one answer from the given options.
3. The application **checks the answer and updates the score**.
4. Questions continue until the quiz is completed.
5. The **final score is displayed** at the end.

# Installation & Setup

Follow these steps to run the project locally.

### 1. Clone the repository

```bash
git clone https://github.com/akshatsharma-x/Quiz-App.git
```

### 2. Navigate to the project folder

```bash
cd Quiz-App
```

---

# Backend Setup

Go to the backend folder and install dependencies.

```bash
cd backend
npm install
```

Start the server:

```bash
npm start
```

The backend server should now be running locally.

---

# Run the Frontend

Navigate to the frontend folder and open:

```
frontend/index.html
```

You can open it directly in your browser.

---

# Usage

1. Start the backend server.
2. Open the quiz application in the browser.
3. Select answers for each question.
4. Submit your answers.
5. View your final score.
